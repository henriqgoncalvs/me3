# Me3

Describe yourself in 3 things, as simple as that. Fill your info and share your profile with your friends.

## !technologies

> Built with [create-t3-app](https://github.com/t3-oss/create-t3-app)

- Next.js
- Typescript
- TailwindCSS
- tRPC
- Next Auth
- Prisma
- React Hook Form
- HeadlessUI
